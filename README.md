## solaris

A minimalist, text-based post-apocalyptic incremental game built with Next.js, React, and Tailwind CSS for Hack Club's Horizons. You play as an autonomous artifical intelligence on desolate earth, using the wreckage of destruction to escape the near-end of the solar system.

## Tech Stack
- Framework: Next.js 15
- Styling: Tailwind CSS v4
- State Management: Custom React hook engine using ('useGameState')
